# 钢琴模拟器

## 特性

提供多种工具

- [滤波合成器](#谐波合成器) 修改声音的合成
- [音色调节器](#音色调节器) 完全自定义音色
- [传函修改器](#传函修改器) 改变传递函数
- [乐谱编辑器](#乐谱编辑器) 用文本编写乐谱并自动演奏

且适配移动端和黑暗模式

### 谐波合成器

- 声音纯物理合成，没有任何采样
- 使用十二平均律产生音阶，允许自由转调
- 使用 10 个谐波，声音足够真实
- 使用 ADSR 包络，同时提供多种可调参数

### 音色调节器

音色调节器允许完全自定义音色，同时也提供了多种预设

- 金属
- 纯净
- 明亮
- 空灵
- 柔和
- 常规
- 真实

### 传函修改器

传函修改器允许修改传递函数。由于传递函数通常在频率上连续的，不便完全自定义，因此只提供预设和可修改的参数

- 纯延时
- 单回声
- 多回声
- 全通
- 低通
- 高通

### 乐谱编辑器

- 用文本编写乐谱，提供实时渲染
- 能够自动演奏，并在乐谱上产生视觉反馈
- 点击乐谱上的音符，会播放对应的声音
- 播放会从被选中的音符开始
- 可以修改速度、节拍和调号
- 提供多种预设乐谱

## 原理

> 详细的原理可阅读我的科普文章[《音乐的数学原理：从振动弦到现代乐理》](https://juemuren.github.io/MyBlogs/posts/math/%E9%9F%B3%E4%B9%90%E7%9A%84%E6%95%B0%E5%AD%A6%E5%8E%9F%E7%90%86/)

### 合成

琴弦振动发出的声音，在理想情况下是由一系列谐波组成的，其中基波的频率最低为 $f_1$，而其余谐波的频率都是基频的整数倍，频率为 $nf_1$ 的谐波就叫 $n$ 次谐波。以正弦波为例

$$p(t) = \sum_{n=1}^{N}A_n\sin(2\pi n f_1 t)$$

取 $N=10$ 时声音已足够真实。

基于此原理，使用 [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API) 合成声音。为了进一步改善听感，通过一系列指数函数来模拟振幅随时间的变化

- 振幅首先被设置为**静音增益**
- 然后在**起音时间**内，振幅变化到被设定的值
- 接着在**衰音时间**内，振幅衰减到**稳音增益**
- 在音符的持续时间内，振幅维持在稳音增益
- 最后在**释音时间**内，振幅重新回到静音增益

为了更符合物理事实，让声音更真实，高次谐波的衰减和释放会更快，且其稳音增益也会变小。代码中使用 $t_n = \frac{t_1}{\sqrt n}$ 来模拟这种变化。

### 音色

音色由各谐波分量的相对振幅，即 $A_n$ 决定。

各音色预设的具体关系为

| 音色 | 谐波振幅关系                                     |
| ---- | ------------------------------------------------ |
| 金属 | $A_n \propto \frac1n$                            |
| 纯净 | $A_n \propto \frac1{n^2}$                        |
| 明亮 | $A_n \propto \frac1n \|\sin\frac{n\pi}2\|$       |
| 空灵 | $A_n \propto \frac{1}{n^2} \|\sin\frac{n\pi}2\|$ |
| 柔和 | $A_n \propto e^{-\sigma n}$                      |
| 常规 | $A_n \propto \frac1{n^2} \sin(n\pi \lambda)$     |
| 真实 | $A_n \propto \frac1{n^p} e^{-\sigma n}$          |

其中的可调节参数有

- $\sigma$ 衰减率
- $\lambda$ 击弦点
- $p$ 幂指数

### 传函

声音在产生到被人耳接受的过程中，可能发生频域上的畸变，即不同的频率分量其振幅和相位会受到不同程度的影响。

各传函的具体关系为

| 效果   | 幅频特性                                                | 相频特性                                                                            |
| ------ | ------------------------------------------------------- | ----------------------------------------------------------------------------------- |
| 纯延时 | $1$                                                     | $-2\pi \tau f$                                                                      |
| 单回声 | $\sqrt{1 + \alpha^2 + 2\alpha\cos(2\pi\tau f)}$         | $-\arctan(\frac{\alpha\sin(2\pi\tau f)}{1 + \alpha\cos(2\pi\tau f)})$               |
| 多回声 | $\frac1{\sqrt{1 + \alpha^2 - 2\alpha\cos(2\pi\tau f)}}$ | $-\arctan(\frac{\alpha\sin(2\pi\tau f)}{1 - \alpha\cos(2\pi\tau f)})$               |
| 全通   | $1$                                                     | $-2\pi\tau f - 2\arctan(\frac{\alpha\sin(2\pi\tau f)}{1 - \alpha\cos(2\pi\tau f)})$ |
| 低通   | $\llbracket f \le f_c\rrbracket$                        | $0$                                                                                 |
| 高通   | $\llbracket f \ge f_c\rrbracket$                        | $0$                                                                                 |

其中的可调节参数有

- $\tau$ 延迟时间
- $\alpha$ 衰减系数
- $f_c$ 频率阈值

### 乐谱

- 使用 [ABC Notation](https://abcnotation.com/) 编写乐谱
- [abcjs](https://www.abcjs.net/) 提供了解析文本和渲染乐谱的功能
- 使用自定义的合成器演奏乐谱，因此需要处理解析后得到的庞大对象
- 动画功能借助渲染乐谱后得到的对象和回调函数实现
